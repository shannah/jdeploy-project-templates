# jDeploy Template Picocli

## About this Project

This project was generated using the Picocli CLI project template from [jDeploy](https://www.jdeploy.com), a tool to deploy Java desktop apps on Windows, Mac, and Linux.

To create your own application based on this template, open jDeploy, select "Create New Project", select the "Picocli CLI Maven Starter" template, and complete the wizard.

## Requirements

* Java 21

## Run
```
./mvnw exec:java
```

## Build

```
./mvnw package
```

### Deploy

**Short Version:**
Open project in [jDeploy](https://www.jdeploy.com), and press "Publish"
